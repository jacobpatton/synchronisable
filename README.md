# Synchronizable

Provides base fuctionality (models, DSL) for AR synchronization with external resources (apis, services etc)

## Installation

Add this line to your application's Gemfile:

    gem 'synchronizable'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install synchronizable
